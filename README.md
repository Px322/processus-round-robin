# Processus Round Robin

## Description
- Simulation de l'algorithme d'ordonnancement **Round Robin** en Python.
- Gestion de plusieurs processus avec un identifiant (PID) et une durée d'exécution.
- Utilisation d'un **quantum** pour répartir équitablement le temps processeur.
- Suivi de l'état des processus : Prêt, En cours d'exécution, Suspendu, Terminé.
- Exécution cycle par cycle jusqu'à la fin de tous les processus.

## Fonctionnalités
- Création de processus avec une durée d'exécution définie.
- Changement automatique de l'état des processus.
- Réinsertion des processus non terminés dans la file d'attente.
- Affichage de l'ordre d'exécution des processus.
- Test de l'algorithme avec différents quantums.

## Technologies
- Python 
- Programmation orientée objet
